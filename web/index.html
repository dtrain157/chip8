Hello, CHIP-8!

A CHIP-8 emulator written in Rust, and comipled to WebAssembly to run in the browser.


INSTRUCTIONS
    Select a ROM, and click RUN to power up the emulator.
    Enable the step-through checkbox to step through the programme a single instruction at a time.
    The programme listing and the contents of the registers are displayed below the screen.
